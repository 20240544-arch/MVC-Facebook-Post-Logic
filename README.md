# MVC Facebook Post Logic 🚀

A simple implementation of the **Model-View-Controller (MVC)** design pattern in Java.

## 📝 Description
This project simulates a social media post where users can add comments. It demonstrates the separation of concerns by dividing the application into three interconnected parts:
- **Model**: Manages the data and logic (Post & Comment).
- **View**: Handles the output/display format.
- **Controller**: Mediates between the Model and the View.
